字幕重命名批处理脚本

发表于 2021-01-01 更新于 2021-05-05 阅读次数： 本文字数： 2.9k 阅读时长 ≈ 3 分钟

将字幕ass全部重命名为与视频mkv一致

前言

最近在看数字图像处理的一些经典思想和算法，重构了一些底层并做了一些应用，简单的数字图像处理已经有点入门了，进一步打算尝试一下视频的简单数字图像处理。

我去字幕组论坛搞了一些相关资料，顺手下了一份钢之炼金术师的BD rip打算边看边练手。手头有一份字幕，但是PotPlayer无法自动匹配对应字幕文件(想想也不太可能)，于是决定把字幕ass全部重命名为与视频mkv一致。几十个文件人工做太浪费时间，决定顺手写个批处理的脚本。

需求

有以下源文件:

处理方式：

  1. 重命名
    [POPGO][Fullmetal_Alchemist][TV_2009][X264_AAC][01](8BF4B9FD).sc.ass–>[VCB-Studio] Fullmetal Alchemist꞉ Brotherhood [01][Ma10p_1080p][x265_flac].sc.ass
  2. 重命名
    [POPGO][Fullmetal_Alchemist][TV_2009][X264_AAC][01](8BF4B9FD).jp.ass–>[VCB-Studio] Fullmetal Alchemist꞉ Brotherhood [01][Ma10p_1080p][x265_flac].jp.ass
  3. 64集Ass(128files)全部按照上述对应关系重命名

分析

  1. 要实现批处理脚本,第一时间想到
    1. linux shell
    2. windows bat
    3. python script
  2. python 实现要使用到
    1. os.listdir遍历目录
    2. shutil.copyfiles复制文件
    3. re.search搜索子串

实现

python:（ASS_[POPGO]2[VCB-Studio]_FullmetalAlchemist.py）
