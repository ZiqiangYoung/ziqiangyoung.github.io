总结

可以看出这个框架的易用性很高，如果搞个图形化配置工具几乎就完全可以吊打某些老牌博客的主题了。配置的解读暂时到此为止，评论系统不太稳定就不搞了，之后有可能会让师弟研究一下如何实现和移植。
